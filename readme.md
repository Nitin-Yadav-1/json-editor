

# JSON Editor
A desktop application made with PySide6 to edit json files.

JSON 'array of objects' is not supported.


## How to run
1. Clone the repo
2. Create and activate virtual environment
3. Install pyside6
    > pip install pyside6
4. run
   > python app.py
